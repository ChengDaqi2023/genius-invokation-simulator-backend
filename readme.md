# Genius Invokation Simulator Backend

----

这里是原神(Genshin Impact)的《七圣召唤》模拟器，是参考原神3.3版本的“七圣召唤”玩法重新实现的后端(服务端)，包括所有的原神内的游戏内容，并拓展一些米哈游没有做的内容。~~不过这个后端只能让你在Postman/ApiFox/Terminal里对局~~

## Features

- [ ] 游戏基本玩法
  - [ ] 角色与技能
  - [ ] 圣遗物与武器
  - [ ] 场景和伙伴
  - [ ] 召唤物
  - [ ] 元素反应
- [ ] 游戏拓展玩法
  - [ ] 创建对局
  - [ ] 匹配对局
  - [ ] 游戏内通信
  - [ ] 观战模式
  - [ ] 比赛模式
  - [ ] 作战记录
  - [ ] 自定义规则
- [ ] 自定义Mod支持
  - [ ] Go支持
  - [ ] Lua支持
- [ ] 多种通信协议连接
  - [ ] websocket接口
  - [ ] http/https接口
  - [ ] udp/kcp接口
  - [ ] rpc接口
- [ ] 分布式支持
  - [ ] 自动化部署
  - [ ] 服务注册与服务发现
  - [ ] 服务端负载均衡
- [ ] 管理功能
  - [ ] 公告与通知
  - [ ] IP追踪/封禁
  - [ ] QPS/TPS限制器
